CHAPTER HANDOFF
  Type: multiple-choice
  Written: 2026-04-25

  The Ferret is 11 meters from the Object's surface, perfectly functional, pointed at what might be a door. The Object redirected it there. It didn't destroy it. It didn't block it. It aimed it.

  **A.** Follow the redirect — advance the Ferret toward the seam and see what happens when it gets closer. The Object is trying to show you something. You'd be a fool not to look.

  **B.** Override the heading and hold position — you want to spend more time scanning the exterior before committing to an approach. You're in control of this probe, not whatever is down there.

  **C.** Pull the Ferret back to the surface. You have enough data to know this is real and significant. Time to burn for the relay, file the claim, and let this become someone else's problem to figure out — properly, with backup.
